## Source-Of-Truth Notes

- R102 is the current final live preflight composition surface.
- R103 records approval intent; it does not authorize execution.
- R104 records dry-run evidence; it does not make the system live-ready.
- R105 defines protocol prerequisites; it does not place orders.
- R106 reports activation readiness for a future phase; it does not enable execution.
- R107 is design only and creates no execution authority.
- R108 is a cockpit and intent ledger only; it cannot place orders or override R106.
- R109 hardens R108's cockpit and sacred button semantics; it remains intent-only and non-executing.
- R110 is a burn-down planner over R102-R109; it writes a report ledger and remains non-executing.
- R111 is a draft prerequisite-clearing phase; it is not order placement and must not create execution authority.
