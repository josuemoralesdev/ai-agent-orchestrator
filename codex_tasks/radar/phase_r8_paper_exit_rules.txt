# Radar Phase R8 — Paper Exit Rules

## Goal

Add configurable paper-only exit rules so Hammer Radar positions can close by take-profit, stop-loss, or max-hold logic instead of only stop-loss.

This phase improves strategy evaluation without adding live trading.

Preserve backward compatibility with existing NDJSON records.

## Exit Logic

For open paper positions:
- long stop if candle low <= stop_price
- short stop if candle high >= stop_price
- long take-profit if candle high >= take_profit_price
- short take-profit if candle low <= take_profit_price
- max-hold closes after configured candle count if neither stop nor TP occurs

If stop and take-profit both occur in same candle:
- fail conservative
- close by stop first
- document this
