ANCHOR INTERACTION RULES:
For each candle after anchor is available, compute:
- close_distance_pct = (close - anchor) / anchor * 100
- high_distance_pct
- low_distance_pct
- touched_anchor:
  true when low <= anchor <= high
- near_touch:
  true when abs(close_distance_pct) <= near_touch_threshold_pct
- above_anchor:
  close > anchor
- below_anchor:
  close < anchor
- cross_up:
  previous close < previous anchor and current close > current anchor
- cross_down:
  previous close > previous anchor and current close < current anchor
- rejection_down:
  high touches or rises above anchor, close below anchor
- rejection_up:
  low touches or falls below anchor, close above anchor
- reclaim:
  cross_up after being below anchor
- loss:
  cross_down after being above anchor

Default thresholds:
- near_touch_threshold_pct = 0.15
- outcome windows = 1, 3, 5, 10, 21 candles
- success threshold pct = 0.10
- adverse threshold pct = 0.10

OUTCOME MAPPING:
For each anchor event:
- direction hypothesis:
  - rejection_down / loss / cross_down => short bias
  - rejection_up / reclaim / cross_up => long bias
  - touch / near_touch => neutral context until paired with signal origin
- compute future close return
- compute MFE in hypothesized direction
- compute MAE against hypothesized direction
- compute simple_success / simple_failure preview
- no leverage, no fees, no slippage, no position sizing
