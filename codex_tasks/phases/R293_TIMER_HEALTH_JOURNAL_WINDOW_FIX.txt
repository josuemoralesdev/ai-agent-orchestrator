       - stale_documentation_warning_ignored_for_current_health

  3. Default recent window:
     Use a safe default of 10 minutes or last 120 lines with timestamp filtering if reliable.
     Prefer journalctl with a valid expression:
       --since "10 minutes ago"
     Do NOT use invalid "now - 3 min".

       journalctl -u hammer-autonomous-trigger-scheduler-dry-run.service --since "10 minutes ago" --no-pager
       journalctl -u hammer-autonomous-trigger-scheduler-dry-run.service -n 240 --no-pager

     Forbidden:
       sudo
       systemctl start
       systemctl stop
       systemctl restart
       systemctl enable
       systemctl disable
