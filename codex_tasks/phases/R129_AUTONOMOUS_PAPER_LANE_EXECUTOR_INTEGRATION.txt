Supported args:

- --lane-key <lane_key>
- --all-lanes
- --confirm-paper-integration "<exact phrase>"

Exact confirmation phrase:
I CONFIRM AUTONOMOUS PAPER LANE INTEGRATION ONLY; NO REAL ORDER; NO BINANCE CALL.

Behavior:
1. Without --record-paper:
   - status: PAPER_EXECUTOR_INTEGRATION_PREVIEW
   - no integration ledger write
   - no paper execution write
   - show candidate decisions and blockers

2. With --record-paper but missing/wrong confirmation:
   - status: PAPER_EXECUTOR_INTEGRATION_REJECTED
   - no integration ledger write
   - no paper execution write
   - rejection_reason: missing or invalid paper integration confirmation

3. With --record-paper and exact confirmation:
   - status: PAPER_EXECUTOR_INTEGRATION_RECORDED or PAPER_EXECUTOR_INTEGRATION_PARTIAL
   - may write integration record
   - may write scheduler tick if --record-scheduler-tick
   - may write autonomy decisions if --record-decisions
   - may write paper execution records for eligible decisions
   - never place real orders
   - never create exchange payloads
   - never call Binance
   - never mutate env
